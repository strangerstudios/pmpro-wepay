== Description ==

IMPORTANT NOTE: This plugin is not yet complete. This was developed as part of a webinar on how to start a gateway add on for Paid Memberships Pro. If you are interested in helping to finish this plugin, please contact info@paidmembershipspro.com or reach out on GitHub.

== Installation ==

1. Make sure you have the Paid Memberships Pro plugin installed and activated.
1. Upload the `pmpro-wepay` directory to the `/wp-content/plugins/` directory of your site.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to Memberships -> Payment Settings in your WordPress dashboard to complete the WePay settings.
